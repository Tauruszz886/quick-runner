# tools

`tools/` stores maintainable helper scripts for this workspace.

Keep source code, small templates, and script README files here. Do not put downloaded toolchains, Python virtual environments, converted CAD files, or one-off experiment outputs in this directory.

Current groups:

- `cad/`: CAD/DXF maintenance scripts.
- `zlj_editor_scene/`: Eggitor editor scene creation, verification, and component-index export scripts.

Runtime game code lives under `ts_src/`. Local tool working files live under `.tools/`.
